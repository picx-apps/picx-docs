快速开始 ​

新建 GitHub 仓库 ​

创建 一个用来存储图片的GitHub 仓库，仓库属性必须选Public 。

点击链接 https://github.com/new 快速新建仓库


新建 GitHub Token ​

创建 一个带有 repo 权限的 GitHub Token 。

点击链接 https://github.com/settings/tokens/new 快速新建 Token

注意：新生成的 Token 只会显示一次，请妥善保存，如有遗失，重新生成即可。


绑定 GitHub 仓库 ​

使用 Chrome 或 Edge 等现代浏览器进入 PicX 图床官网，填写前面新建的 GitHub Token，完成图床和仓库绑定。

  1. 填写 Token，自动获取该用户下的所有 Public 仓库。

  2. 在选择仓库的下拉列表中，选择刚刚新建的那个仓库作为图床。

  3. 选择一种目录方式（目录即仓库里存放图片的文件夹）。

    • 新建目录：需手动输入一个新目录。

      使用 / 可创建多级目录，例如：输入A01/A02 表示创建二级目录，最大支持创建到五级目录。

    • 根目录：图片将存储在仓库根目录下。

    • 自动目录：自动生成日期格式 yyyyMMdd 的目录。例如：20230203。


上传图片 ​

  • 可以通过 拖拽图片 到上传区域进行上传。

  • 可以通过 复制图片，然后 粘贴 到上传区域进行上传。

    粘贴图片快捷键 ctrl + v 或 command + v

  • 可以通过点击上传区域，选择图片 进行上传。

点击 上传 按钮或使用快捷键 ctrl + s 或 command + s 开始上传图片


复制图片链接 ​

图片上传成功后，会自动生成图片链接并复制到系统剪贴板，通过快捷键 ctrl + v 或 command + v 就可以拿去粘贴使用了。
